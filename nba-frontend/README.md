# NBA Analytics Frontend

Expo React Native client for searching NBA players/teams and rendering stat trends.

## Commands

- `npm install`
- `npm run start`
- `npm run android`
- `npm run ios`
- `npm run web`
- `npm run lint`

## Environment

Set the backend base URL for API calls:

- `EXPO_PUBLIC_API_BASE_URL` (default: `http://localhost:3000`)
